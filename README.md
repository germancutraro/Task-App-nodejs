# Task App - Node.Js - Express and MongoDB

Rest Task app using nodejs

## Instalation

Install all dependencies

```
npm install
```

MongoDB is a on-demand database, just run the app.

Run app

```
npm start
```

## Built With

* Node.Js
* Express.Js
* MongoDB
* Bootstrap
